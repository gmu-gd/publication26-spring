Project 1: Book Design

Description

Using the provided manuscript, design a cover and interior for the book using the provided specifications. This project will have you explore the basics for book and other long format projects.

We will explore the principles of editorial design, InDesign page set-up (facing pages, parent pages, columns, margins, gutter, folios and automatic page numbering, running heads), font pairings, pt. size / leading, etc.

Requirements

Page Size

  • 5" x 8" page size
  • .5" spine width

Jacket Design

  • Front Cover – Title and author
  • Spine - Title, Author, and Stillhouse Logo
  • Back Cover - Book description, Stillhouse logo and socials, author socials, bar code

20 Page Booklet

  • Page 1: Front Cover
  • Page 2: Blank
  • Page 3: Title Page
  • Page 4: Copyright/Colophon
  • Page 5: Dedication
  • Page 6: Blank
  • Page 7: Table of Contents
  • Page 8: Blank
  • Pages 9–18: Book Content
  • Page 19: Blank
  • Page 20: Back Cover

Things to Keep in Mind

  • Avoid trite designs
  • Do not use illustration as a default
  • Always better to go too far than not go far enough
  • Printing takes time, as does trimming

Objectives

  • Implementing a typographic system
  • Creating designs that reflect a concept
  • Building craft skills
  • Building InDesign skills
  • Creating digital mockups

Checkpoints

  1. five digital cover concepts, printed and trimmed in color
  2. two refined jacket concepts, three interior spread concepts
  3. refined full cover, full interior designed
  4. printed, trimmed, and bound final draft, digital mockups

Final Submission

  • Turned in on time (10/2/25)
  • Printed trimmed, and bound 20 page booklet
  • Printed and trimmed jacket
  • Digital mockups
  • PDF of jacket
  • PDF of book (cover, interior as spreads, back cover)
  • Photos of physical book
  • All images and pdfs on Canvas
