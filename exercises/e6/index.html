Exercise 6: Styles

Details

Download the E6-styles.indd file from Canvas. Rename all the paragraph styles to something useful and change the typefaces for them. Using no more than two, relatively neutral typefaces, change the font for each paragraph style. You do not need to change any other parts of the paragraph style.

Submission Requirements

Submit a PDF as spreads with no crop marks and a screenshot of your renamed paragraph styles on Canvas.
