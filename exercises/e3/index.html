Exercise 3: Hierarchy

Details

Create three different 11x17 posters using this text. You must use all the provided text and cannot add or remove any of the text. In some way, highlight the rule you find most important of the ten. Follow the typeface requirements below for each poster.

  • The first poster will only use one of the typefaces you chose for E2.
  • The second poster can use two of the typefaces you chose for E2.
  • The third poster can use three of the typefaces you chose for E2.

Each poster should use the same grid but be different in layout. Try different ways to create hierarchy in each poster and consider the types of information in the text and how they might be treated.

Submission Requirements

Submit a single PDF of all three posters on Canvas. Print your favorite design and bring it to the next class.
