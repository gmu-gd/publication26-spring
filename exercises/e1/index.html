Exercise 1: Setting It Up

Details

Create a single page, 5"x8" InDesign document with half inch margins and an eighth inch bleed. Add grid to it that is five columns and eight rows with the gutters as an eighth inch. Make sure the guides are fit to the margin, not the page.

Submission Requirements

Print your single page centered on a letter sized sheet with the guides visible and with crop marks and bleed marks in color or black and white. Also save your single page as a PDF with the guides visible and with crop marks and bleed marks. Submit that PDF on Canvas. Bring the printed version to class next week.
